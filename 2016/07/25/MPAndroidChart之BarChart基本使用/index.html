X轴的设置

xAxis.setGranularity(float granularity)
官当的解释是 Set a minimum interval for the axis when zooming in. The axis is not allowed to go below that limit. This can be used to avoid label duplicating when zooming in.
翻译为: 在缩放时设定最小间隔为轴线。该轴是不允许走低于该限制。这可用于避免标签翻炒在缩放时。
表示还是不怎么弄，上图自己看吧

xAxis.setGranularity(1f);

xAxis.setGranularity(2f);

可以发现在X轴的值有所不同，设置为1的时候，每个树状图下都有值，当设置为2的时候，树状图下的值则是各一个显示一个
